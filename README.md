# Markdown Editor

[![GitHub Pages](../../actions/workflows/github-pages.yaml/badge.svg)](../../actions/workflows/github-pages.yaml)

The personal markdown editor of [Stefan Kühnel](https://stefankuehnel.com).

## Get Started

To get started with the project, follow these steps:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/stefankuehnel/editor.git
   cd editor
   ```

2. **Install dependencies:**

   ```bash
   task install
   ```

3. **Run the development server:**

   ```bash
   task dev
   ```

4. **Open your browser:**

   Navigate to `http://localhost:5173` to see the markdown editor in action.
